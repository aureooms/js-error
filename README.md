[js-error](http://aureooms.github.io/js-error)
==

Error objects for JavaScript. Parent is
[js-library](https://github.com/aureooms/js-library).

```js
import {
	IndexError ,
	KeyError ,
	NotImplementedError ,
	TypeError ,
	ValueError ,
} from '@aureooms/js-error' ;
```
